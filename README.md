# ig-to-youtube

A tool for uploading media from Instagram to YouTube automatically.

## Structure

- `.github/workflows/upload.yml`: Github Actions workflow for automated uploads.
- `requirements.txt`: Python dependencies.
- `sources.json`: Instagram sources configuration.
- `index.json`: Index of uploaded videos.
- `uploader.py`: Main upload script.
- `get_yt_refresh_token.py`: Script for getting YouTube OAuth token.
- `seo_utils.py`: SEO helpers for optimizing video metadata.
- `.gitignore`: Files/folders to exclude from git.

## Usage

See individual scripts and workflow for setup instructions.
